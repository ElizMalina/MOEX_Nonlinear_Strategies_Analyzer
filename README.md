# Анализ финансовых данных на MOEX. Нетрадиционные методы. 

## 📊 Стратегии

### 1. 🎃 Стратегия Хэллоуина 
Классическая календарная стратегия, основанная на рыночной поговорке:
- **Покупка**: с 1 ноября по 30 апреля
- **Продажа**: с 1 мая по 31 октября

### 2. 🌙 Лунная стратегия
Торговля на основе фаз Луны:
- **Покупка**: на растущую Луну 
- **Продажа**: на убывающую Луну

### 3. 🌡️ Погодная стратегия (Санкт-Петербург)
Торговля на основе температуры в Санкт-Петербурге:
- **Покупка**: когда среднесуточная температура > 10°C
- **Продажа**: когда температура ≤ 10°C
